# Employee Attrition Prediction

This project aims to predict whether an employee will leave the company using machine learning techniques.

## Algorithm
- Data preprocessing
- Feature selection
- Model training using Random Forest
- Prediction

## Dataset
- IBM HR Analytics Dataset
- 1470 records
- Features: Age, Salary, Job Role, Department, Job Satisfaction

## Tools Used
- Python
- Pandas
- Scikit-learn

## Model Performance
- Algorithm: Random Forest Classifier
- Accuracy: 85.84%

## Output
The model predicts whether an employee will Stay or Leave based on input features.
